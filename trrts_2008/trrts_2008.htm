Item Description

Internal feature number.
















Shape

Feature geometry.
















ROUTE

Route number
















NAME

Name of route. Often simply the name of the street associated with route
















COUNTY

County or counties where routes are located
















YEAR

Year the routes were updated
















DESCRIPT

Field added by GeoPlan based on ROUTE and NAME fields. If there is a non-0 ROUTE field value and no NAME field value, then the DESCRIPT field value will be "ROUTE #". If there is a NAME field value and no ROUTE field value then the DESCRIPT field value will be "NAME". If there is a ROUTE field value and a NAME field value then it will be "ROUTE # - NAME". If there is no ROUTE field value or the value of the ROUTE field is 0 and no NAME field value then the DESCRIPT field will be "NO DESCRIPT AVAILABLE".

DATA LINEAGE SUMMARY: 








GeoPlan received this dataset from Albert Gan via email in October of 2008. 
The data was received in individual shapefiles by county and was in the following 
projection: Albers, NAD 83 HARN. The datasets were merged using the merge 
tool in arcgis.

- Added field NAME based on most appropriate name field provided
- Added COUNTY field based on county where route is located
- Added year field based on currency of data
- Added DESCRIPT field based on ROUTE
- Trimed leading whitespaces in DESCRIPT field 
- Added FGDLAQDATE based on date downloaded from source
- Upcased all fields
- Original file received was named routes.zip which contained 26 
  folders containing routes for specific areas
Process Date: 20081120


Updated DESCRIPT field by doing the following:
If there is a non-0 ROUTE field value and no NAME field value, then the 
DESCRIPT field value will be "ROUTE  #". If there is a NAME field value and no 
ROUTE field value then the DESCRIPT field value will be "NAME". If  there is a 
ROUTE field value and a NAME field value then it will be "ROUTE # - NAME". If 
there is no ROUTE  field value or the value of the ROUTE field is 0 and no NAME 
field value then the DESCRIPT field will be "NO  DESCRIPT AVAILABLE".
